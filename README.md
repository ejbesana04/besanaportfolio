# Portfolio Website

Simple static portfolio built with HTML, CSS, and JavaScript.

To view locally:

1. Open `index.html` in your browser, or
2. Run a simple local server (recommended):

```bash
# Python 3 (from project root)
python -m http.server 8000
# then open http://localhost:8000
```

Files added:
- css/styles.css
- js/script.js

Customize the content by editing `index.html`.
